# Fastapi_boooks_store_endpoints


 
The Online Bookstore needs a complete API to support:
 
1. User Management
   - User registration
   - User login/authentication
   - User profile management
 
2. Book Catalog
   - Browse all books
   - Search books by title, author, genre
   - Filter books (by price, rating, availability)
   - View book details
 
3. Shopping Cart
   - Add books to cart
   - Update quantity
   - Remove from cart
   - View cart
 
4. Order Management
   - Place order
   - View order history
   - Track order status
   - Cancel order
 
5. Reviews & Ratings
   - Submit book review
   - Rate books (1-5 stars)
   - View all reviews for a book
